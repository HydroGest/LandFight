# LandFight — 快节奏国战核心插件

> **设计背景**
> 
> 传统国战服务器往往存在“游戏周期过长”、“玩家在线时间不统一”等弊端，容易导致后期发育严重不公、新玩家难以融入。
> 为了解决这一痛点，我们倾力打造了一款**高节奏、纯技术、速战速决**的全新轻量化国战玩法插件！

---

## 核心玩法规则

### 1. 战场与地图生成
* **地图边界**：游戏将在标准的 **1500 × 1500** 的有限随机世界中展开，倒逼玩家频繁接触，爆发高频遭遇战。
* **资源点位**：为了简化繁琐的占地过程，系统会在地图内随机预生成 **30 个核心据点**，作为双方争夺的战略焦点。

### 2. 据点占领与大本营（首都）机制
* **占领据点**：玩家前往据点，对其中的特定 **【羊毛】** 方块点击 **鼠标右键**，即可为己方阵营占领该据点。
* **设立首都（大本营）**：玩家可以通过 **潜行（Shift）+ 右键** 点击已占领据点内的羊毛，将该据点强行升格为己方的“大本营/首都”。
* **复活机制**：大本营将作为团队成员的唯一指定安全复活点。保护大本营的安全是维持战线的核心要素。

### 3. 战术迁移与沦陷规则
* **攻占与偷家**：敌方阵营可以随时攻占你的普通据点。同样，如果防守空虚，你的首都也可以被敌方直接强占。
* **沦陷惩罚**：一旦己方首都沦陷（被敌方占领），全队成员将 **无法继续复活**，陷入极度危险的绝境。
* **动态战略（改都）**：在首都彻底沦陷以及你真正死亡之前，只要手里还握有其他普通据点，你可以随时通过潜行右键 **任意更改大本营位置**，实现金蝉脱壳与战略转移！

### 4. 胜负结算条件
| 结算类型           | 具体规则描述 |
|:---------------| :--- |
| ** 常规时间结束**    | 单局游戏时长严格控制在 **45 分钟**。时间耗尽后，系统将自动结算各队伍的据点占领数量，**占城多者获胜**。 |
| ** 提前终局（剃光头）** | 当某一阵营的 **大本营全部沦陷**，且该阵营的 **所有在线成员全部死亡** 时，游戏将立即提前终止，直接判定另一方获得压倒性胜利。 |
